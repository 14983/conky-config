# Conky Configuration

---

Display **Uptime**, **Load Average**, **Time**, **Memory Usage**, and **CPU Usage** on your desktop.

Usage:

```bash
git clone https://github.com/14983/conky-config.git
cp conky-config/conkyrc ~/.conkyrc
```

- The default font is `JetBrains Mono`, which can be changed as needed. A monospace font is recommended.
- Modify the number of cores (default is 10) to suit your requirements.
